Why the name

Named for a rare crystal.
Made with the same clarity.

Phosphophyllite is one of the rarest minerals on Earth — prized for its clean, transparent blue-green facets. We borrowed the name as a standard to live up to: every product we ship should be as easy to see through as the ingredient panel on the back.

Founded in 2014, we now formulate and manufacture across four categories — beverages, snacks, personal care and home care — from a single set of sourcing principles.

How we work

Clarity, from source to shelf.

Every ingredient we use is traceable to a named supplier. Every package is designed to be recycled, not just labeled recyclable. Every formulation is reviewed by an independent food-and-cosmetic safety panel before it reaches a shelf.
